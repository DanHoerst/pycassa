logger – Pycassa Logging¶

Logging facilities for pycassa.

class pycassa.logger.PycassaLogger(level='info', logger_name='pycassa')¶

Creates a new PycassaLogger.

This class uses the Borg design pattern to acheive singleton-like sharing.

Parameters:
  • level – the logging level for the logger
  • logger_name – the name that will be used to call logging.getLogger()
